# Authentication

Implementing User authentication using Passport.js

###  Dependencies
1. Passport
2. Passport-Local
3. Passport-Local-Mongoose
4. Passport-Facebook
5. Express-Session

### Running Locally 

1.   Clone the repository 

  ```git
  git clone https://github.com/namandangi/auth
  ```
2.   Run the project 

  ```git
  node app.js
  ```
